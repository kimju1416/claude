Gemini 결과가 있나요?

Gemini에서 복사한 수업 자료가 있다면 아래 "0. 시작점 선택" 칸에 바로 Ctrl+V로 붙여넣으세요!
없다면 그냥 아래 섹션들을 선택하면 됩니다.

Claude Design 한도 절약 팁: 이 빌더의 핵심은 한 방에 좋은 결과를 뽑는 거예요. 입력이 자세할수록 재시도 횟수가 줄어 한도가 절약됩니다. 필수는 2번(강의 형식)뿐, 나머지는 비워두면 AI가 자동 보완합니다.
V4.0 핵심 전략: Claude Design의 강점인 파일 업로드/웹 캡처/디자인 시스템을 최대한 활용하도록 프롬프트가 설계되었습니다. 처음부터 만들지 말고 기존 자료를 재활용하세요!

생성된 프롬프트를 Claude Design 캔버스에 붙여넣으세요 (Ctrl+Enter 단축키 지원)

Gemini 2분 + Claude 3분 = 총 5분. 기존 수업 준비 대비 10배 이상 빠릅니다.

🎨 Claude Design 10가지 강점

기존 PPT 제작 방식과 완전히 다른 핵심 강점입니다.

1

PPTX export 최적화

생성 후 원클릭으로 .pptx 파일 다운로드. PowerPoint나 Keynote에서 미세 조정 가능. 편집 가능한 텍스트/도형으로 나와서 원본 수정이 자유롭습니다. 한국 교사에게 가장 실용적인 핵심 기능!

✨ 빌더 활용: 8번 섹션의 "PPTX" 칩 선택 → 프롬프트에 "Optimize for PPTX export" 자동 삽입됩니다.

2

기존 파일을 디자인으로 변환

DOCX/PPTX/XLSX/PDF를 그대로 업로드하면 자동으로 시각 자료로 변환. 긴 강의 스크립트(워드)를 붙여넣으면 슬라이드로, 엑셀 데이터를 넣으면 차트 중심 인포그래픽으로 만들어줍니다.

✨ 빌더 활용: 0번 섹션의 "기존 PPTX 업로드/DOCX 업로드/XLSX 데이터" 칩 활용.

3

Web Capture로 레퍼런스 가져오기

웹사이트 URL을 제공하면 해당 사이트의 요소를 참고해 슬라이드에 반영. 자기가 좋아하는 디자인 톤의 사이트 URL을 주면 그 분위기를 담아낼 수 있어요.

✨ 활용 팁: Claude Design에서 직접 레퍼런스 URL을 제공하면 돼요. Toss, Notion 등 자기 맘에 드는 사이트를 자유롭게.

4

브랜드 디자인 시스템 자동화

한 번 브랜드를 등록하면 이후 모든 프로젝트가 내 컬러/로고/폰트/컴포넌트를 자동 상속. 교내 자료, 연수 자료, SNS 카드뉴스가 모두 같은 DNA 공유. 매번 스타일 지정할 필요 없이 일관성 유지!

✨ 빌더 활용: 5번 섹션 "디자인 시스템" - CORE 기능! 한 번만 설정하세요.

5

정적 → 인터랙티브 프로토타입 전환

일반 슬라이드를 클릭 가능한 퀴즈, 드래그 앤 드롭, 플립 카드로 전환. 학생 자기주도 학습, 온라인 강의, 영상 녹화에 강력. HTML로 export하면 링크 하나로 학생들에게 배포 가능.

✨ 빌더 활용: 7번 섹션 "인터랙티브 프로토타입" 칩 + 8번에서 HTML export 선택.

6

Canva로 원클릭 전송

초안은 Claude Design에서 빠르게, 최종 마감은 Canva에서. 한 번 클릭으로 Canva 편집 가능한 파일로 전환. Canva의 풍부한 템플릿/일러스트/스톡 이미지와 결합 가능.

✨ 빌더 활용: 8번 섹션 "Canva" Export 선택.

7

인라인 코멘트 & 슬라이더 미세 조정

캔버스에서 특정 요소를 클릭해 인라인 코멘트 달아 정밀 수정 가능. 여백/색/레이아웃은 커스텀 슬라이더로 실시간 조정. 큰 변경은 채팅, 작은 수정은 인라인으로 나누면 한도 절약!

✨ 활용 팁: 전체 구조는 생성 후 그대로 두고, 특정 슬라이드만 인라인 수정하세요.

8

조직 URL 공유 & 협업

디자인을 조직 URL로 즉시 공유. private/view-only/edit access 세 가지 권한. 동료 교사와 함께 Claude와 그룹 대화하며 수정 가능. 공동 발표자와 자료 제작할 때 유용.

✨ 빌더 활용: 8번 섹션 "URL 공유" Export 선택.

9

Claude Code 핸드오프

디자인이 마음에 들면 "handoff bundle"로 패키징해서 Claude Code에 전달 → 실제 작동하는 웹앱으로 구현. 인터랙티브 학습 자료, 웹 기반 퀴즈 플랫폼 등 코드 수준 결과물까지 가능.

✨ 고급 활용: 8번 섹션 "Claude Code 전달" 선택. 주현님 바이브 코딩 경험과 찰떡!

10

음성/영상/3D 등 프론티어 기능

Claude Design 공식 소개에 따르면 voice, video, shaders, 3D, built-in AI를 활용한 code-powered 프로토타입 제작이 가능합니다. 단순 슬라이드를 넘어 멀티미디어 요소가 포함된 자료 실험에 활용해볼 수 있어요.

✨ 프론티어 기능: 아직 실험적이에요. 일반 교사 업무에선 덜 쓰이겠지만 차별화 요소로 알고 있으면 좋아요.

붙여넣기 → 엔터 → Export

Ctrl + V 후 엔터. AI가 슬라이드를 한 번에 생성. 우상단 Export 버튼으로 PPTX 다운로드!

💰 왜 토큰(한도)이 절약되나요?

① 왕복 대화 제거

AI가 "청중은 누구죠?", "시간은 몇 분이죠?" 같은 질문을 아예 안 하게 만들어요. 프롬프트 상단에 "AVOID asking clarifying questions" 지시가 이미 박혀있습니다.

② 재생성 횟수 감소

AI가 한 번에 맞는 결과를 내놓으니 "다시 만들어줘" 요청이 줄어요. Claude Design은 한 번 생성할 때마다 큰 토큰을 쓰기 때문에 이게 결정적입니다.

③ 컨텍스트 누적 방지

Claude Design은 대화가 길어질수록 이전 캔버스 상태 전체를 매번 다시 전송해요. 수정 요청이 10번이면 10번 다 누적됨. 첫 판에 잘 뽑으면 이 누적이 줄어요.

A

🔰 처음 써보는 경우 - 빠른 체험

B

⚡ 실전 사용 - 기존 자료 리뉴얼 가장 많이 쓸 패턴

예전 PPT를 새로 고치고 싶을 때 가장 강력한 방법. 처음부터 만드는 것보다 5배 빨라요.

  1. 0번 섹션 → "📊 기존 PPTX 업로드" 칩 클릭
  2. 1번 섹션 → 청중 규모·사전 지식·환경 선택
  3. 2번 섹션 → 강의 형식 선택 (필수)
  4. 3번 섹션 → 스토리텔링 구조 (예: PREP, 골든서클)
  5. 6번 섹션 → 디자인 스타일 하나 선택
  6. 8번 섹션 → "PPTX" 선택
  7. 생성 & 복사
  8. Claude Design에서 "+" 클릭 → 기존 PPTX 파일 첨부
  9. 복사한 프롬프트 붙여넣기 → 엔터
  10. → 기존 자료가 새 디자인으로 변환되어 나옴!
C

🎨 브랜드 일관성 만들기 - 한 번만 설정하면 평생 활용

한 번 설정하면 이후 모든 자료가 같은 톤으로 나오는 마법. 주현님처럼 여러 자료를 만드는 사람에게 필수!

  1. 빈 프로젝트로 Claude Design 접속
  2. 빌더의 5번 섹션 → "📐 브랜드 시스템 양식" 칩 클릭
  3. 양식에 본인 정보 채워넣기 (로고, HEX 컬러, 폰트 등)
  4. 생성 & 복사 → Claude Design에 붙여넣기
  5. 브랜드 로고 파일도 함께 첨부
  6. Claude가 디자인 시스템 자동 구축
  7. → 이후 만드는 모든 자료가 이 브랜드 톤으로 자동 생성!

  • • 템플릿: "인터랙티브 학습"
  • • HTML export 필수 (PPTX는 정적 변환)
  • • 학생에게 URL 공유 → 자기주도 학습

⚠️ 꼭 기억할 5가지

  1. 발표자 정보는 직접 입력하세요. AI가 채우지 않습니다. 0번·1번 섹션 양식 칩 참고.
  2. 한 방에 끝내세요. "이것도 추가해줘" 재요청은 한도를 빨리 갉아먹어요. 이 빌더로 미리 다 넣으세요.
  3. 간단한 자료는 Claude Design 쓰지 마세요. 일반 채팅 Artifacts로 충분한 것들은 한도 아끼기.
  4. 인터랙티브는 HTML, 정적은 PPTX. 7번에서 클릭 퀴즈 같은 거 넣었으면 8번에서 HTML export 꼭 선택.
  5. 기존 파일 있으면 무조건 업로드. 처음부터 만드는 것보다 5배 빠릅니다.

🔥 바로 해볼 수 있는 첫 실험

감 잡으려면 이 순서로 해보세요:

  1. 빌더 열기
  2. [Claude Design 강점] 탭 열어서 쭉 훑어보기 (3분)
  3. [추천 템플릿] 탭에서 "우수사례 발표" 클릭 (자동 복사됨)
  4. claude.ai/design 접속 → 새 프로젝트
  5. Ctrl+V → 엔터
  6. 결과 나오면 우상단 Export → PPTX 클릭
  7. 다운받은 파일 열어보기 → "아, 이거구나!" 감 잡기 완료

📑 섹션별 상세 가이드

아래는 좌측 빌더의 각 섹션을 어떻게 채우는 게 좋은지 상세 설명입니다.

0

시작점 선택이 가장 중요합니다 (NEW)

V3.0의 핵심 혁신입니다. 처음부터 만들지 마세요! 기존 자료가 있다면 파일 업로드부터 시작하는 게 정확도 5배 빠름.
- 예전 PPT를 새 톤으로 리뉴얼 → "기존 PPTX 업로드"
- 강의 스크립트/원고 있음 → "DOCX 업로드"
- 설문·성적 데이터 있음 → "XLSX 데이터"
- 레퍼런스 디자인 사이트 있음 → "Web Capture"

1

청중 분석이 디자인의 절반입니다

"코딩 강의"보다 "코딩 강의 (90분, 코딩 무경험 교사 30명, 일반 회의실)"처럼 규모+사전지식+동기+환경까지 적으세요.
발표자 정보는 직접 입력. AI가 임의로 채우지 않습니다.

2

강의 형식부터 골라야 톤이 잡힙니다 (필수)

교사 연수 vs 학생 수업 vs 행사 발표는 디자인 톤이 완전히 다릅니다. 이 칸만 필수입니다.

3

스토리텔링 구조가 발표의 절반입니다

같은 내용이라도 순서에 따라 청중 반응이 천차만별. 짧은 보고는 PREP, 영감 강연은 골든서클, 사례는 STAR, 책 강연은 픽사 스토리, 학생 수업은 도입-전개-정리, SNS는 AIDA.

4

슬라이드 패턴으로 구조를 단단하게

"표지 → 학습목표 → 목차 → 핵심 개념 3개 → 사례 → Q&A → 마무리" 흐름을 미리 정해두면 결과물이 단단해집니다. 학습목표 슬라이드는 한국 교실 거의 필수!

5

디자인 시스템이 V3.0의 킬러 기능 (NEW)

한 번 브랜드를 등록해두면 모든 후속 자료가 자동으로 내 톤. 매번 스타일 지정하지 않아도 일관된 결과물!
- 학교 공식 모드 / 개인 브랜드 모드 구분 가능
- 기존 PPT/웹사이트에서 참고 가능

6

디자인 스타일은 1개만

여러 스타일 섞으면 혼란. 한 강의 = 한 스타일 원칙. 디자인 시스템이 있으면 이 칸은 비워도 됩니다.

7

인터랙티브는 HTML로, 정적은 PPTX로

클릭 퀴즈·드래그·플립 카드 선택했다면 반드시 HTML export 또는 URL 공유로 배포해야 인터랙션 보존됨. PPTX는 정적으로 변환되니 주의!
접근성·인용·발표자 노트는 프로와 아마추어의 결정적 차이입니다.

8

용도에 맞는 Export 선택

- 오프라인 발표, 동료 수정 → PPTX
- 인쇄 핸드아웃 → PDF
- 웹 배포, 인터랙티브 → HTML / URL 공유
- 추가 디자인 편집 → Canva
- SNS 업로드 → PNG
- 실제 웹앱으로 구현 → Claude Code 핸드오프

슬라이드 매수 = 발표 시간 ÷ 슬라이드당 시간

• 키노트/연사: 슬라이드당 2-3분 (60분 → 20-30장)

• 일반 발표: 슬라이드당 1-2분 (30분 → 15-20장)

• 워크숍/실습: 슬라이드당 3-5분 (90분 → 20장 + 실습)

• SNS 카드뉴스: 1슬라이드 = 1메시지 (5-10장)

• 학생 수업: 슬라이드당 2-3분 + 활동 시간 별도

Claude Design 한도 절약 황금 규칙

첫 프롬프트에 모든 걸 압축: 이 빌더가 도와드려요.

3.

5.

디자인 시스템 한 번만 구축: 이후 모든 프로젝트 자동 상속.
